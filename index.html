СЕЛЕКТОРИ

Селектор Приклад Пояснення до прикладу
active a:active

Вибірка активного посилання.
attribute [target]

Обирає всі елементи, у котрих вказано атрибут target.
all *

Обирає всі елементи
before p::before

"Додає що-небудь перед контентом елемента"

checked input:checked

"Обирає кожний елемент" input " який має значення checked"

focus input:focus

"Обирає елемент" input ', який є у фокусі'

id #firstname

Вибирає елемент з індетифікатором id="firstname"
hover a:hover

Обирає посилання на яке наведено курсор
link a:link

Обирає всі посилання по яким ще не переходили
not :not(p)

Вибірка елементів,котрі не містять вказаний селектор.
